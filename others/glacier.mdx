<Info>
**Sürekli çalışacak.**

Kullandığım sunucu: **Ubuntu 22.04**

Minimum: \
CPU with 1+ cores \
2GB RAM \
4 MBit/sec download Internet service

Recommended: \
Fast CPU with 2+ cores \
4GB+ RAM \
8+ MBit/sec download Internet service
</Info>


1. Bir EVM cüzdanı oluşturun ve private key çıkarın.
2. https://thirdweb.com/opbnb-testnet adresine gidin. Ana cüzdanınızı bağlayın ve bir hesap oluşturun. Cüzdan adresinizi yapıştırın ve opBNB ağından tBNB talep edin.
3. https://chainlist.org/chain/5611 adresinden opBNB ağını cüzdanınıza ekleyebilirsiniz.
4. Ana cüzdanınızdaki tBNB leri yeni oluşturduğunuz cüzdan adresine opBNB ağından gönderin.

Sunucuya geçelim

#### Screen oluşturalım

```bash
screen -S glacier
```

#### Eğer docker yüklü değilse yükleyin

```bash
sudo apt update -y && sudo apt upgrade -y
for pkg in docker.io docker-doc docker-compose podman-docker containerd runc; do sudo apt-get remove $pkg; done

sudo apt-get update
sudo apt-get install ca-certificates curl gnupg
sudo install -m 0755 -d /etc/apt/keyrings
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg
sudo chmod a+r /etc/apt/keyrings/docker.gpg

echo \
  "deb [arch="$(dpkg --print-architecture)" signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu \
  "$(. /etc/os-release && echo "$VERSION_CODENAME")" stable" | \
  sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

sudo apt update -y && sudo apt upgrade -y

sudo apt-get install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
sudo docker run hello-world
```

#### Docker ile node çalıştıralım

> Oluşturduğunuz cüzdanın private keyini **YOUR_PRIVATE_KEY yazan kısma YOUR_PRIVATE_KEY yazısını silip yapıştırın.**

```bash
docker run -d -e PRIVATE_KEY=YOUR_PRIVATE_KEY --name glacier-verifier docker.io/glaciernetwork/glacier-verifier:v0.0.1
```

Logları görmek için
```bash
docker logs -f glacier-verifier
```

Screenden **CTRL + A + D** çıkabilirsiniz.
